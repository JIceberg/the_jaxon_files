Introduction to Limits

What is a limit? It is a point on the function f(x) to which f approaches.

Take, for instance, the function h(x) = x + 2.
If the limit is at the point x = 3, then you must look at the point to which h approaches from either side. In this case, the limit is at the point ( 3, 5 ). This is because, as we increase the values of x nearer to 3, the y-value approaches 5. As we decrease the values of x nearer to 3, the y-value approaches 5 as well. Because the point is approached from both directions, the limit does exist, and exists at y = 5.

Consider the function f(x) = ( x − 1 )⁄( x − 1 ) .
If the limit is at the point where x = 1, then look at the point to which f approaches from either direction. In this case, the function approaches the point ( 1, 1 ), which is undefined. That's because it is a hole in the graph (see Alg. II/Geo. B). Even though the point is a hole, the limit does not have to be a point that exists on the graph. Therefore, our limit is ( 1, 1 ).

Imagine the function g(x) = {x2 (@ x ≠ 2), ⅓ (@ x = 2)}.
What is the limit of g(x) as x approaches 2? Remember, the limit does NOT need to be a defined point on the graph; it only needs to be a point that g is approaching. In this case, the limit is ( 2, 4 ) instead of the defined point ( 2, 1 ).



Characteristics of Limits

What is the limit when f approaches x = 4 for the function f(x) (seen above)?
In this case, the limit would be ( 1, 1 ) because the function is continuous. If the function were discontitnuous, then the limit would not exist.

That considered, look at this function: p(x).
It's clearly discontinuous, so what would the limit be at x = 8? The answer is that there is no limit, because there is no point to which p approaches from either directions at x = 8. However, at x = 0, the limit does exist—at the point ( 0, 0 ).

Limits of One Direction
Although functions do not have a limit at points of discontinuity, there can still be a limit for each piece of the graph.

In p(x), there is no existing bidirectional limit at x = 8. But what about the limit as x approaches 8 from the left?
The notation for a left-side limit is x → 8-. This means that p is approaching 8 from all x-values less than 8 (including -100, 0, 4, 7.9, 7.99, 7.99999). In this case, the left-side limit is at ( 8, 4 ), the undefined point.

What is the right-side limit? Is it the same as the left-side limit? The short answer is no. The reason for this is that p does not approach ( 8, 4 ) from the right. Instead, it approaches the point that DOES exist at x = 8: ( 8, -2 ).
The notation for the right-side limit is x → 8+. This means that p is approaching 8 from all x-values greater than 8 (including 50, 10, 9, 8.1, 8.01, 8.0001).

Limits of Unique Functions
How unique is a function's limit at any one point? Not at all. An infinite number number of various functions can be made from a limit.

In a continuous function, the limit can be determined thusly: limx → n f(x) = f(n).
Discontinuous functions must have, at some point, a nonexistent limit. If the discontinuity occurs at x = d, then the limit is the following: limx → n f(x) = f(n), where n ≠ d.

Limit Properties
What are some of the basic properties that enable the ability to simplify future limit problems?

1) Addition/Subtraction
Imagine that limx → c f(x) = L1. If limx → c g(x) = L2, then what is limx → c (f(x) + g(x))?
The limit of a sum or difference is the sum or difference of the limits.
In this case, we can rewrite the question as limx → c f(x) + limx → c g(x), and then we can substitute in the limits of the functions: L1 + L2.

2) Multiplication/Division
Keeping the limits of f and g the same, what is limx → c (f(x) × g(x))?
The limit of a product is equal to the product of the limits; therefore the question can be rewritten as limx → c f(x) + limx → c g(x), and then we can substitute in the limits of the functions: L1L2.
Similarly, dividing functions results in the quotient of their limits. However, the limit that is the divisor must not equal zero, or the result will be undefined.

When finding the limit of a constant times a function (e.g. limx → c kf(x)), it is the same as the product of the constant and the limit of the function.

3) Exponents
What is limx → c [f(x)]e, keeping the limits the same?
If e is real, then the result is equivalent to (L1)e.

Unbounded Limits
What does it mean when the limit of a function is infinity as it approaches a certain x-value?

Consider this: limx → 0+ q(x), where q(x) = 1⁄x.
The answer? There is no limit. Not because the two sides don't converge at the same point—the limit is one-sided anyways—but because q approaches infinity at x = 0. Therefore, we refer to the limit as unbounded.

Determining Limits from a Table of Values

Consider the following table:
x g(x)
1.9 3.61
1.99 3.96
1.9999 3.9996
2 ⅓
2.0001 4.0004
2.01 4.04
2.1 4.41

What is a reasonable estimate for limx → 2 g(x)? Despite there being an existing point at ( 2, ⅓ ), the limit does not have to be defined by the function. Instead, it only needs to be a point that is approached by g, which is 4 in this case.

Look at this table:
x p(x)
7.9 3.96
7.99 3.985
7.9999 3.999
8 -2
8.0001 -2.0001
8.01 -2.01
8.1 -2.1

What is the estimated value of limx → 8 p(x)? Regarding the points on the table above, there is no point to which both sides of the function approach. Therefore, there is no bidirectional limit for p(x).
What is the estimated value of limx → 8+ p(x)? This question is asking for a single-side limit, rather than a bidirectional one. So, the limit exists at the point to which p nears from the right (as indicated by the superscript '+' after the 8). Thus, the right-side limit at x = 8 is -2.

Oscillating Limits
Consider the function r(t) = sin(1⁄t).
What is limx → 0 r(t)? Let's set up a table:
t r(t)
-2(7π)-1 1
-2(65π)-1 -1
-2(635π)-1 1
0 undef.
2(635π)-1 -1
2(65π)-1 1
2(7π)-1 -1

As you can see, there are infinitely many oscillations, never nearing any specific number. Therefore, we refer to these kinds of limits as oscillating. Similarly to unbounded limits, oscillating limits do not truly exist.
